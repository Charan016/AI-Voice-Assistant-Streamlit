# AI Voice Assistant (Streamlit)

This is a voice-enabled AI assistant built using Streamlit, Speech Recognition, and Groq LLM.

## Features
- 🎤 Voice input using microphone
- 🤖 AI responses using LLM (Groq API)
- 🔊 Text-to-Speech output
- 💬 Chat history support
- 🎛️ Voice control (Male/Female)

## Tech Stack
- Python
- Streamlit
- SpeechRecognition
- pyttsx3
- Groq API

## How to Run

1. Clone the repo:
   git clone https://github.com/your-username/AI-Voice-Assistant-Streamlit.git

2. Install dependencies:
   pip install -r requirements.txt

3. Add your API key in `.env`:
   GROQ_API_KEY=your_key_here

4. Run the app:
   streamlit run app.py
