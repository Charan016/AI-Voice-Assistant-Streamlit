# AI Voice Assistant (Streamlit)

This is a voice-enabled AI assistant built using Streamlit, Speech Recognition, and Groq LLM.

## 🚀 Features

* 🎤 Voice input using microphone
* 🤖 AI responses using Groq LLM
* 🔊 Text-to-Speech output
* 💬 Chat history support
* 🎛️ Voice control (Male/Female)

## 🛠️ Tech Stack

* Python
* Streamlit
* SpeechRecognition
* pyttsx3
* Groq API

---

## ▶️ Run Locally

1. Clone the repository:
   git clone https://github.com/your-username/AI-Voice-Assistant-Streamlit.git

2. Navigate into the folder:
   cd AI-Voice-Assistant-Streamlit

3. Install dependencies:
   pip install -r requirements.txt

4. Create a `.env` file and add:
   GROQ_API_KEY=your_key_here

5. Run the application:
   streamlit run app.py

---

## 🌐 Deployment (Streamlit Cloud)

You can deploy this app online using Streamlit Cloud:

1. Push your code to GitHub
2. Go to https://share.streamlit.io
3. Connect your GitHub repository
4. Select `app.py`
5. Add your GROQ_API_KEY in Streamlit secrets
